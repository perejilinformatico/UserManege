## About

UserManage is a user management system built with **Blazor** and **PostgreSQL**.

The application allows administrators to create, edit, delete and manage users through a modern web interface.

## Features

- Create users
- Edit user information
- Delete users
- User listing
- PostgreSQL database integration
- Responsive UI

## Tech Stack

- Blazor
- C#
- ASP.NET Core
- Entity Framework Core
- PostgreSQL
